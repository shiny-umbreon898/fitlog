# fitlog
Fitness tracking web app using Flask, React



## Installation
1. Install Python 3.8 or higher
2. Clone the repository
3. Install dependancies using pip - run the following command in the terminal
	pip install flask flask-cors flask-sqlalchemy flask-migrate


4. Run the app - run the following command in the terminal
	python app.py

http://127.0.0.1:5000/ local development server
